About the Project

The Distro Transparency Index is a project created by a Linux enthusiast. For privacy reasons, I have chosen not to disclose my identity. While I'm not a professional in the open-source world, Linux is my great passion, and this project stems from that enthusiasm.

If you have any questions, suggestions, or need further clarification, please don't hesitate to contact me at: contact@distrotransparencyindex.org.

Furthermore, if you're interested in collaborating on the project, your contribution would be greatly appreciated. Whether it's improving the methodology, updating distribution data, or contributing to the website's code, all help is welcome. Please contact me using the email address above to discuss how you could participate.
